3.4.1 RZ, NRZ, AMI, Manchester y Otros Códigos de Línea

Introducción

Los códigos de línea son técnicas utilizadas en telecomunicaciones para representar datos digitales mediante señales eléctricas. Cada método tiene características particulares relacionadas con el ancho de banda, la sincronización, la detección de errores y la inmunidad al ruido.

RZ (Return to Zero)

El código RZ es un código con retorno a cero, donde la señal vuelve al nivel cero dentro del intervalo de cada bit. Esto mejora la sincronización, pero aumenta el ancho de banda requerido.

  • RZ neutral: El bit 0 se representa con un nivel +A y regresa a 0V a mitad del bit; el bit 1 se mantiene en nivel bajo.
  • RZ polar: El bit 0 se representa con una transición de +A a -A; el bit 1 no presenta transición.
  • RZ bipolar: El bit 1 no tiene transición; el bit 0 usa niveles alternados con transición.

NRZ (Non Return to Zero)

En los códigos NRZ la señal no regresa a cero durante el intervalo del bit. Son más eficientes en ancho de banda, pero presentan problemas de sincronización en secuencias largas.

  • NRZ-L: Voltaje positivo representa 0 y voltaje negativo representa 1.
  • NRZ-I: El bit 1 se representa con una inversión de voltaje; el bit 0 no provoca cambio.

Código AMI (Alternate Mark Inversion)

El código AMI representa los bits 1 mediante pulsos positivos y negativos alternados, mientras que los bits 0 se representan sin voltaje.

  • No permite más de 15 ceros consecutivos.
  • Facilita la detección de errores.
  • Elimina la componente DC.

Aplicaciones del AMI

  • Redes WAN
  • Transmisión infrarroja
  • Códigos de barras
  • Reducción de diafonía en cables telefónicos

Pseudoternaria

La codificación pseudoternaria es similar a AMI, pero en este caso los ceros alternan su polaridad y los unos se representan sin voltaje.

Manchester

La codificación Manchester combina los datos y el reloj en una sola señal, proporcionando autosincronización.

  • Transición de negativo a positivo → bit 1
  • Transición de positivo a negativo → bit 0

Es ampliamente utilizada en redes Ethernet como 10Base5 y 10Base-F.

Manchester Diferencial

La codificación Manchester diferencial indica el valor del bit mediante la presencia o ausencia de una transición al inicio del intervalo del bit.

  • Bit 1: no hay transición al inicio
  • Bit 0: hay transición al inicio
  • Siempre existe una transición a mitad del bit

Es más robusta frente al ruido y funciona incluso si la señal se invierte.

B8ZS (Bipolar 8-Zero Substitution)

El código B8ZS se utiliza en sistemas T1 para evitar largas secuencias de ceros. Cuando aparecen ocho ceros consecutivos, se introducen violaciones bipolares.

La sustitución es: 000V B0VB, donde:

  • V: Violación bipolar
  • B: Transición bipolar válida

HDB3 (High Density Bipolar 3)

El código HDB3 es una mejora del AMI que reemplaza secuencias de cuatro ceros por patrones especiales para mantener la sincronización y eliminar la componente DC.
